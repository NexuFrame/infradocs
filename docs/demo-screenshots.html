Full-Text Search

Instantly search across all devices, IPs, VLANs, and notes. The generated search index makes finding anything fast.

Dark Mode Default

Easy on the eyes for late-night troubleshooting. The dark theme is perfect for NOC and datacenter environments.

📱

Mobile Friendly

Access your docs from your phone in the server room. Responsive design works on any screen size.

🖨️

Print Ready

Generate PDF-ready documentation for audits, onboarding, or compliance. Clean print CSS included.

Works Offline

Static HTML files — no server required. Host on any web server, NAS, or open locally.

✅

Data Validation

Catch errors before they become problems. Validates IP addresses, MAC addresses, CIDR notation, and cross-references.

Print-Optimized Output
The generated site includes print-friendly CSS. When you print (or "Print to PDF"), the navigation hides and the layout optimizes for paper — perfect for compliance documentation or physical runbooks.
