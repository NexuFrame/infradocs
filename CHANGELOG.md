# Changelog

All notable changes to InfraDocs Generator will be documented here.

## [1.0.0] — 2026-05-26

### Added
- CLI: `infradocs build`, `infradocs validate`, `infradocs init`
- YAML, JSON, and CSV input support
- 11 HTML page templates: index, site, device, vlan, subnet, printer, vendor, circuit, change-log, emergency, plus base
- Built-in JavaScript search (no external dependencies)
- Dark-mode CSS, print-friendly stylesheet
- Sample data: 3 sites, 12 devices, 8 VLANs, 5 subnets, 4 printers, 5 vendors, 3 circuits
- 4 pytest test files
- Full documentation in docs/
