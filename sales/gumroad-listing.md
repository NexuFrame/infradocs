## Description

Tired of out-of-date network diagrams and scattered spreadsheets? InfraDocs Generator reads your YAML and CSV files and produces a clean, searchable, offline-ready HTML portal — covering devices, VLANs, subnets, printers, vendors, circuits, change logs, and emergency runbooks.

**Perfect for:** solo IT admins, MSPs managing client networks, school/university IT, manufacturing plant IT.

### What you get:
- Python CLI: `infradocs build --input data --output site`
- 11 HTML templates (dark mode, mobile-friendly, print-ready)
- Built-in search — no server required, works fully offline
- Sample data with 3 sites, 12 devices, 8 VLANs and more
- 4 pytest test files
- Full docs (installation, configuration, troubleshooting, FAQ)
- MIT licensed — use in commercial environments

## FAQ

**Does it need internet?** No. 100% offline once installed.
**Windows compatible?** Yes. Python 3.8+ on Windows, Mac, Linux.
**Can I customise the templates?** Yes — standard Jinja2.
**What if my data format is different?** The loader is extensible. See docs/input-formats.md.
**Is there a free trial?** The sample-data folder gives you a fully working demo out of the box.
